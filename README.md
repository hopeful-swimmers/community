# Community

This is the program's permanent forum. The conversation lives in [Discussions](https://github.com/hopeful-swimmers/community/discussions), not here.

## What you'll find in Discussions

- **Announcements** — Wave Kickoffs, scheule changes, Demo Day dates.
- **General** — Open conversation between Wave members and anyone curious.
- **Q&A** — Questions about the program (not for code-specific debugging — use a `help-wanted` issue on your own parcipant repo for that).
- **Show and tell** — Demo Day write-ups and updates from past participants.
- **Ideas** — Proposals to improve the curriculum or the program.

Anyone on the internet can read Discussions; only org members can post.

For the program overview and how it runs, see the [org landing page](https://github.com/hopeful-swimmers).
